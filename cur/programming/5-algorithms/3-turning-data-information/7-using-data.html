In this assginment, you will build a program similar to what you did in the lab. From the CORGIS Datasets Project use the Weather dataset. Note that each line of the dataset contains weather information from the week, not a single day. The dataset covers a single year, 2016.

Requirements

In this lab you will write a program that can answer each of the following questions or perform the requested operation. Your program should simply have a button for each item that the user can click to view the results. This means when your program starts it will display a list of buttons to press. The user should be able to the button menu after viewing the results.
  1. What was the average maximum temperature in the state of Texas?
  2. What was the highest average wind speed in Honolulu, Hawaii?
  3. How many weeks were there without rainfall in Tulsa, Oklahoma?
  4. What was the total (not total of weekly average!) rainfall for the year in Lansing, Michigan?
  5. Create a bar chart that shows a max temperature range and the number of times a city fell in the range. Use increments of 5 degrees. (Note: each city should be counted 52 times, once for each week.
  6. Do the same as the previous, but only use cities in California.
  7. Determine the maximum rainfall value for each city (the largest weekly value from the entire year). Create a bar chart that shows the maximum rainfall range and the number of cities that fell in the range. Each city should only be counted once in the graph.
