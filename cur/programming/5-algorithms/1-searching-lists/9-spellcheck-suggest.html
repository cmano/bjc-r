In this assignment you will build on the spellchecker program. Your program will compare a binary search and linear search approach to spell checking text. After finding an incorrectly spelled word, the program report how many word comparisons you made in order to determine if the word was contained in the list. It will also make suggestions on a possible replacement.

Requirements

  1. Ask the user for input. The user will type as much as they want.
  2. Check each word in the input with the 100,000 words(sorted) list.
  3. When a word is not found in the list, display a message to the user with the following:
    • Number of checks performed to determine the word is not in the list using the binary approach and the linear approach.
    • Suggestion of a word they may have meant. Just select the word before or after the spot where the misspelled word would have been alphabetically in the list.
  4. Let the user decide if they want to use the suggested word. If they do, then swap the old word with the new word.
  5. After completing the entire list of words, display the following to the user:
    • Number of misspelled words
    • How many times the user chose to use the new word
    • Print the entire text the user entered, but with the new words as applicable
