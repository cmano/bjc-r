Checking for Ties

On this page, you'll check for a tie game, that is, whether there are no more empty squares but neither player has won.

  1. Click for a hint about checking for ties.
    Develop a way to determine if the game has tied, and if so, make the program notify the players.
  2. Play a couple of games to test the tie-checking feature, and fix any bugs.

    3. You can work on this project more in Lab 25: Tic-Tac-Toe with a Computer Player.
