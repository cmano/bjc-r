Submit Snap Homework

After completing a Snap lab or assignment, you'll need to submit your program to be graded.
  1. Click the File menu icon in the toolbar,
  2. Select "Export Project".
  3. Rename the project using the format Last-First-Lab-#.xml.
  4. Add the file to the folder that you will will submit.

After you upload anything to Canvas, you should always download it and verify that your submission is correct. You would be surprised how often the incorrect file is uploaded.
