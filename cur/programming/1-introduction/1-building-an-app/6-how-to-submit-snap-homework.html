Submit Snap Homework

This is your first example of submitting an assignment. Assignments are completed individually. This means if you're completing your labs in pairs, you need to stop working together and each of you must do the assignment by yourself. For some assignments you'll actually submit the code you created in Snap!. Follow these instructions for all Lab Checks and Assignments that require you to submit code.

Download Snap! Code File

Use these steps for every Snap! programming submission.
  1. Click the File menu icon in the toolbar,
  2. Select "Export Project".
  3. Rename the project using the format Last-First-Assn-#.xml.
    • Example: Smith-Jane-Assn-1.xml
    • For Lab Checks you will change "Assn" to "LC"

Lab 1 Assignment

Typically an assignment will require work and understanding of the concepts you just learned. A more complete set of requirements will be given here or on the Canvas page where you'll submit the assignment. A rubric will also be available on Canvas.
  1. Export and save your program using the approriate name
  2. Upload the file to Canvas.
