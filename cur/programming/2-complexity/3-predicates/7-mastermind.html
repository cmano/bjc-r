Lab 8 Assignment

In this project, you will begin to build the game Mastermind™. In this game, the player tries to guess the sequence of colors selected by the computer.

Game Overview

  • The computer will select a random sequence of four colors selected from six color choices.
  • The colors may not be repeated.
  • The player guesses a sequence. ex: rgoy (red, green, orange, yellow)
  • The computer reports two numbers
    1. The number of colors in the correct position.
    2. The number of correct colors (colors that appear in the sequence) but are in an incorrect position.
  • The player continues to guess until the correct sequence is discovered.

Getting Started


Look at the scripts. There are six sprites, four for the player's guess and two for the feedback to the guesser. Each sprite already has all necessary costumes. There are starter scripts in the stage, the first color, and the first number (correct position) sprites. Make sure to use those to see how things work and make your life easier. Try to get all of the sprites to show the new guess first. Then you can worry about adding the code to check the users guess.

Feedback to the guesser is two numbers (as shown in each row of the game in the image above):

  • How many colors are in the correct positions? (The goal is to have all four correct.)
  • How many correct colors are in incorrect positions?

Requirements

  1. Clicking the green flag starts the game.
  2. Prompt the user for their name. Save this so you can use it to make the messages personal.
  3. Prompt the user for a guess with an appropriate message, which includes their name. Use the initials of the color for the guess. The starter script has an example. However you need to modify this to include the user's name in the prompt.
  4. Display the guess on the screen along with the number of colors in the correct position and number of correct colors in an incorrect position.
  5. Repeat until the player guesses correctly. If the user makes so many guesses the entire screen gets filled, don't worry about it. Just hit the stop button and play it again.
  6. Show a message congratulating the winner. Include their name and the total number of guesses they used. Your win message can look however you'd like. It does not need to match the one shown above. You may want to show a new sprite who says the congratulations message. Don't forget to include their name in the message.
