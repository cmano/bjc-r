Lab 8 Assignment

In this project, you will begin to build the game Mastermind™. In this game, tries to guess the sequence of colors selected by the computer.

Game Overview

  • The computer will select a random sequence of four colors selected from eight color choices.
  • The colors may not be repeated.
  • The player guesses a sequence. ex: rgoy (red, green, orange, yellow)
  • The computer reports two numbers
    1. The number of colors in the correct position.
    2. The number of correct colors (colors that appear in the sequence) but are in an incorrect position.
  • The player continues to guess until the correct sequence is discovered.

Getting Started


Look at the scripts. There are six sprites, four for the player's guess and two for the feedback to the guesser. You will need to add additional color sprites.

Feedback to the guesser is two numbers (as shown in each row of the game in the image above):

  • How many colors are in the correct positions? (The goal is to have all four correct.)
  • How many correct colors are in incorrect positions?

Requirements

  1. Ask the user for their name
  2. Prompt the user for a guess with an appropriate message. Use the initials of the color for the guess.
  3. Display the guess on the screen along with the number of colors in the correct position and number of correct colors in an incorrect position.
  4. Repeat until the player guesses correctly.
  5. Show a message congratulating the winner. Include their name and the total number of guesses they used.
